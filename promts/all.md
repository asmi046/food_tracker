# Бот для мессенджера MAX для контроля питания

Нужно сделать бот на основе бибилиотеки maxhub/max-bot-api. Назначение бота: ведение дневника питания.

## Описание логики работы бота

- Пользователь регистрируется в боте для него заводится учетная запись с минимальными данными
- после регистрации пользователью открываются следующие возможности:
  - отправляет фото продукта оно фиксируется в базе данных с учетом даты и времени. Фотография сжимается и кладется в хранилище. После фото отправляется по api в нейронку она распознает продкут и выводит структурированные данные в которых будет наименование продукта и его БЖУ
  - поотзователь может отправить комманду для подсчета калорийности за день.
